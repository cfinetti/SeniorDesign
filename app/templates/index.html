Parking Availability

Here you can see the real-time parking availability on campus. The map is updated in real-time to reflect the current status of each parking lot.

Clicking on a marker will allow you to see the name of the lot and the current number of estimated parking spots available.

The markers correspond to the portion of spaces left as follows:

100-75% (Green)
50-25% (Yellow)
25-0% (Red)
